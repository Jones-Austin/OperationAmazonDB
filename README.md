# Mock Amazon Project New

## Overview

This project is a mock Amazon-style web application built for CMPS480.

It includes:

* Frontend (React / Vite)
* Backend (Node.js / Express)
* Database (MySQL)

## Features

* Product listing
* Search and filtering
* Shopping cart (add, view, remove)
* Database schema with users, products, orders

## Database

The database was designed using MySQL Workbench.

SQL file:

```
docs/database-implementation.sql
```

## How to Run

### Backend

```
cd server
npm install
node index.js
```

### Frontend

```
cd client
npm install
npm run dev
```
